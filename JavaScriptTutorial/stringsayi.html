JavaScript String

JavaScript dizeleri metin depolamak ve değiştirmek için kullanılır.

JavaScript Dizeleri

JavaScript dizesi, tırnak içine yazılmış sıfır veya daha fazla karakterdir.

ÖRNEK:var x = "John Doe";
ÖRNEK:var carName1 = "Volvo XC60"; // Double quotes

var carName2 = 'Volvo XC60'; // Single quotes

IP uzunluğu

Bir dizenin uzunluğunu bulmak için built-in lengthözelliğini kullanın:

ÖRNEK:var txt = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";

var sln = txt.length;

JavaScript Numaraları

JavaScript'in yalnızca bir tür numarası vardır. Sayılar ondalıklı veya ondalıksız olarak yazılabilir.

ÖRNEK:var x = 3.14; // A number with decimals

var y = 3; // A number without decimals

Çok büyük veya çok küçük sayılar bilimsel (üslü) gösterimle yazılabilir:

ÖRNEK:var x = 123e5; // 12300000

var y = 123e-5; // 0.00123

Sayı ve Dize Ekleme

UYARI !! JavaScript hem ekleme hem de birleştirme için + işlecini kullanır. Sayılar eklenir. Dizeler birleştirilir.

ÖRNEK:var x = 10;

var y = 20;

var z = x + y; // z will be 30 (a number)
